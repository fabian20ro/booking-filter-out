Booking.com Hotel Filter

Filter out hotels from search results — works on mobile, no extension needed.

1 Copy the bookmarklet

Tap the button below to copy a small script to your clipboard.

2 Create the bookmark

  1. Open any page in Safari
  2. Tap the Share button (square with arrow)
  3. Tap Add Bookmark and save it
  4. Open Bookmarks (book icon) and find it
  5. Tap Edit, then tap the bookmark
  6. Clear the URL field and paste the copied code
  7. Rename it to "Hotel Filter" and save
  1. Open any page in Chrome
  2. Tap the ⋮ menu > Bookmark (star icon)
  3. Tap ⋮ menu again > Bookmarks
  4. Find the bookmark, tap ⋮ next to it > Edit
  5. Clear the URL field and paste the copied code
  6. Rename it to "Hotel Filter" and save
  1. Open any page in Firefox
  2. Tap the ⋮ menu > Bookmark this page
  3. Go to Bookmarks in the menu
  4. Long-press the bookmark > Edit
  5. Clear the URL field and paste the copied code
  6. Rename it to "Hotel Filter" and save

3 Use it

  1. Go to booking.com and search for a location
  2. Open your bookmarks and tap "Hotel Filter"
  3. A toolbar appears at the bottom of the screen
➕ Save visible hotels
🔍 Dim saved hotels
📋 Copy the rest
🧹 Clear the list
➕ 🔍 📋 🧹 The toolbar appears at the bottom of your screen. Tap the badge to see saved hotel names.
